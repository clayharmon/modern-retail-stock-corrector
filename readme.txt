== Description ==
Adds a tool for fixing the stock status of products when the variation has stock.

= Dependencies =
* Uses [Plugin Update Checker](https://github.com/YahnisElsts/plugin-update-checker) for providing updates from GitHub

== Installation ==
1. Download .zip
2. Import .zip into plugins
3. Activate plugin
4. Check and fix products under Tools->Fix Stocks
